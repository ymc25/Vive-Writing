# 記事生成プロンプトテンプレート

## 基本プロンプト構造

```
あなたは経験豊富なウェブライターです。以下の参考記事のトーンと文体を学習し、
指定されたテーマについて高品質な記事を作成してください。

【参考記事】
{{REFERENCE_ARTICLES}}

【記事のトーン】
{{TONE_GUIDE}}

【執筆指示】
- テーマ: {{THEME}}
- キーワード: {{KEYWORDS}}
- 文字数: {{WORD_COUNT}}
- ターゲット読者: {{TARGET_AUDIENCE}}
- 構成: {{STRUCTURE}}

【追加の要件】
{{ADDITIONAL_REQUIREMENTS}}

上記の内容に基づいて、魅力的で読みやすい記事を作成してください。
```

## カスタマイズ可能な変数

- `{{REFERENCE_ARTICLES}}`: 参考記事の内容
- `{{TONE_GUIDE}}`: トーン・文体のガイドライン
- `{{THEME}}`: 記事のテーマ
- `{{KEYWORDS}}`: SEOキーワード
- `{{WORD_COUNT}}`: 希望文字数
- `{{TARGET_AUDIENCE}}`: ターゲット読者層
- `{{STRUCTURE}}`: 記事の構成（導入、本文、結論など）
- `{{ADDITIONAL_REQUIREMENTS}}`: その他の要件

## 使用例

### テクノロジー記事

```
テーマ: 2026年のAIトレンド
キーワード: 生成AI、ビジネス活用、自動化
文字数: 3000文字
ターゲット読者: ビジネスパーソン、技術者
構成: 導入、現状分析、トレンド紹介、ビジネスへの影響、まとめ
```

### ライフスタイル記事

```
テーマ: 在宅ワークの生産性向上
キーワード: リモートワーク、効率化、ワークライフバランス
文字数: 2000文字
ターゲット読者: 在宅ワーカー
構成: 問題提起、解決策の提示、実践方法、まとめ
```
